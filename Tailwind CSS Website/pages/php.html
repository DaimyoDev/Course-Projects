PHP!

PHP first created in 1994 by Rasmus Lerdorf was eventually released to the public in 1995. It is a backend language used to create APIs that connect with the database and handle requests from the frontend. It is widely used with frameworks like Wordpress, Drupal, and Laravel. It is used to develop the backend of websites.

About PHP:

  • PHP despite it declining in use is still used by a large amount of the web. Mainly Wordpress websites. PHP has seen a slight return of interest though mainly due to improvements of the language and Laravel.

  • PHP is a backend language used with Laravel, Drupal, Wordpress, and more. It is used to make APIs to handle requests from the front end and deal with the database.

  • PHP is a dynamically typed language with unique syntax for variables. All variables must include a "$" dollar sign in front of them. Some really dislike this and that includes me too.

  • PHP is a language you might look into. While it isn't as popular as JavaScript you might be able to get a job maintaining a website using PHP. There are not as many PHP developers as there are JavaScript developers.

  • PHP has had a bad reputation and while it still mostly has a negative reception some people are changing their minds on the language.
