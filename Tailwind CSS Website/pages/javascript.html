JavaScript!

JavaScript was originally invented in 1995 by Brendan Eich. It was created for the web. However, it can be used for general purpose and other applications like desktop with Electron, React Native, Native Script, and more for cross platform development. It is a dynamically typed language which means it does not have type safety. This could lead to potential bugs and errors within a program.

Why Learn Javascript:

  • JavaScript is used in almost every website and is used in every web app!

  • The vast majority of software engineering jobs will require you to know JavaScript! While there are many jobs, many people also know JavaScript so the job hunt may be fierce.

  • As mentioned above, JavaScript can be used for cross-platform development with React Native, Native Script, and more. You can develop desktop applications using Electron.

  • JavaScript has many frameworks built upon it that many employers use in their day to day development and maintenance of their web apps. React being the biggest and most popular one but there are other frameworks like Qwik, Lit, Svelte, Vue, Angular, Solid, AlpineJS, HTMX, and many more!

  • One downside is the JavaScript world is innovating and advancing quickly. New frameworks almost every week it seems like. Although most companies play it safe and usually don't switch frameworks very often. For a while now React has become the major player in the space. Angular, Svelte, and Vue playing smaller roles.

  • JavaScript is a simple language to pick up and it has many advanced features that you don't necessarily need to know to start building websites and applications. Depending on your app's scope you may need to use different features of the language.

  • JavaScript is still being updated and worked on so improvements are happening all the time! Although, some of these changes might require you to update or change old code.
