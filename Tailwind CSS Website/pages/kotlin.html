Kotlin!

Kotlin started developing in 2010 by JetBrains. JetBrains is mainly a company known for their different IDEs. Kotlin was made to be an alternative to Java. It is mainly used for native android development but you can also use it for web development using Ktor. You can also use Kotlin with HTMX!

About Kotlin:

  • Kotlin is the main way we develop native android applications today. We do not use Java anymore. With webpack compose you can also make UIs and apps for IOS using Kotlin!

  • Most jobs for native android development will require knowledge of Android Studio and Kotlin.

  • Kotlin is a statically typed language unlike JavaScript, PHP, and Python. This means we have type safety which can reduce unknown bugs and errors.

  • One downside to Kotlin is that there aren't as many jobs compared to JavaScript and Python. It is mostly app development which is not as popular as it used to be. While app development will never go away completely, the decline of native apps and the rise of cross platform development with React Native and Flutter harm Kotlin's main use case.

  • As a result of lower demand and declining demand there are less developers out there who know Kotlin so the job market may not be as competitive as JavaScript or Python.

  • Kotlin is a much simplier language than Java. It is really a replacement for Java in many ways but still is not as widely used as Java.

  • While Kotlin is meant to be a general purpose language it hasn't found much use yet outside of native mobile development. Although you can use it for web development it isn't used much there.
