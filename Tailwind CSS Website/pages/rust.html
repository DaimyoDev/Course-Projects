Rust!

Rust was created in 2006 by Graydon Hoare but officially released in 2010. Rust is a competitor to languages like C and C++. It is a lower level language compared to JavaScript and Python. It is statically typed and has type safety. It also does not have garbage collection so you will need to manage your own memory.

About Rust:

  • Rust is one of the most beloved languages out there. It is mostly used for backends for websites, desktop applications using Tauri, operating systems, and other low level devices.

  • There aren't as many jobs for Rust but Rust is in demand and while there are some people learning Rust most of them are senior developers. Rust jobs seem to be all over the place.

  • Rust is considered to be a hard language to learn similar to C++. The language it is mostly compared against.

  • Rust can be used for game development using the Bevy engine. An open source game engine that uses Rust! Due to Rust's insane performance it is great for game development and I'm sure it will be used more in the future.

  • You can build websites in Rust using Yew, Web Assembly, HTMX, and Rocket. Although I'm not sure how many people would need to build a website in Rust, you certainly can do so.
