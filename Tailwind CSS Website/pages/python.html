Python!

Python was created in 1991 by Guido Van Rossum. The oldest language on our website so far! Python is a simple language to learn used heavily for general purpose. Web development, machine learning, and data science are the main uses of Python.

About Python:

  • Python is the second most popular language ever. Right behind JavaScript. It has tons of jobs especially in the AI and Data Science field. There are some web development jobs too mainly using either FastAPI or Django.

  • There are tons of jobs for Python but at the same time there are also a lot of people who know Python so competition is fierce! Plus most jobs like data science and machine learning require a masters degree!

  • Python can also be used for game development using pygame but it is more of a fun little hobby and due to Python's poor performance games should not be made using Python.

  • Python is dynamically typed and uses indents for the scope of the language. It is a language beginners often learn when they are completely new to programming! The syntax is incredibly simple.

  • Python is one of the slowest languages out there! It is not compiled but interpreted which makes it a lot slower compared to other programming languages like C, C++, Zig, Go, and Rust.
